# information-retrieval-system

** Ejecuci\'on del proyecto **

Lo primero es entrar al directorio `src`, una vez adentro ejecutas los siguientes comandos

* Consola:

```bash
python3 main.py
```

* Test

```bash
python3 test.py
```

* Visual

```bash
./manage.py runserver
```

Si posees la herramienta makfile, tambi\'en puedes ejecutar los siguientes comandos:

```bash
make # main
make test
make serve
```

** Directorios: **

En resource se encuentra almacenado la infomracion de cada estructura para optimizar
el trabajo, tambi\'en se almacenan datos como son los rating y las consultas para feedback. Se recomienda no eliminar los cluster ya que estos son muy costosos de construir.
